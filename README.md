# cryptpadtab - encrypted notes in a browser tab

This is a toy project to learn more about some modern web APIs, notably
[streams](https://developer.mozilla.org/en-US/docs/Web/API/Streams_API) and
[crypto](https://developer.mozilla.org/en-US/docs/Web/API/Crypto). It is
inspired by [notepadtab](https://github.com/revolter/notepadtab.com) and
[staticrypt](https://github.com/robinmoisson/staticrypt).

The page then consists of a single textarea. The text you enter will be
gzipped, base64-encoded, and then stored in the URL. The URL parameter
`encrypt` enables encryption. The URL parameter `view` displays the content
instead of the textarea. That way you can also serve encrypted websites.
